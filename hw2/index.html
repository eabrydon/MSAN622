Homework 2

Multivariate Data Visualization

Here is a link to the homework 2 code The code is in index.html.

The bubble chart was inspired by The Wealth and Health of Nations. I used the same axes and bubble size - life expectancy, per-capita income, and population as that graph. I color coded the states by region in the US. As this plot is static, unlike the other one, I changed the mouseover functionality a little bit. I also lowered the opacity a bit, as I wanted to use larger bubbles for this chart. There are fewer states in the US than there are countries in the world, so I thought larger bubbles would be appropriate, but there was significant overlap.

For the small multiples, I did a grid scatter plots. I thought this would be interesting because many of the variables in the dataset seem to be heavily correlated. I chose variables that I thought might have correlation. As you can see from the plot, there is a fairly strong negative correlation between life expectancy and high school graduation rate. One design decision I made was to remove grid lines from the plot. I did this because these plots are generally used to visually inspect datasets to see correlations between variables. Grid lines are nice for being able to see exact values, but in this case exact values are more or less irrelevant.
