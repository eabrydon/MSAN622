Dataset Summary

The dataset I plan to use is a subset of the data from the Global Terrorism Database. From their website: "The Global Terrorism Database (GTD) is an open-source database including information on terrorist events around the world from 1970 through 2013 (with annual updates planned for the future). Unlike many other event databases, the GTD includes systematic data on domestic as well as international terrorist incidents that have occurred during this time period and now includes more than 125,000 cases."
