Project Sketch

Early Outline for Data Vis Final Project

  • Planned Tools

    • I plan to use Tableau Public to play with the data and D3 to implement my final visualizations. I will preprocess the data in python as the dataset is fairly large to start with.

  • Planned Techniques

    • The data has longitude and latitude coordinates, so I plan to do a plot to show the which areas of the world have the most terror attacks by plotting the attacks as opaque circles. For the other plots I'll make a line graph showing terror attacks over time, highlighting significant conflicts, and a line plot showing the ratio of people wounded and people killed over time.

  • Planned Interaction

    • I plan to have the heatmap of terror attacks change based on a selection on either of the time series plots, so you can see terror attack density for a chosen time period.
